## Pomodoro Timer App

A simple pomodoro timer app with an integrated
to-do list.

## Features

# Pomodoro Timer

Customizable Short and Long Breaks
Start/Pause/Stop Control Buttons

# To-Do List

Add new tasks
Check task as complete
Delete tasks

# Additional Features

Light/Dark theme toggle
